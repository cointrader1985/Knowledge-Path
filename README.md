# Knowledge-Path
The simulation represents a structured progression system inspired by a general lesson flow, where the player advances through stages representing academic growth from school level challenges toward more advanced university difficulty. Progress is tracked using a simple scoring mechanism based on correct decisions.

The system also references abstract learning materials such as a virtual book, which influences random events during gameplay. The goal is to show how loops, conditionals, and state tracking work in a complete program.

The project is lightweight, easy to compile, and designed for educational experimentation and modification.
